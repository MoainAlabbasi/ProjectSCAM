## 1. الخلفية (Backend)

| التقنية | الإصدار / النمط | الغرض الأساسي | ملاحظات |
| :--- | :--- | :--- | :--- |
| **Python** | 3.11+ | لغة البرمجة الأساسية للتطبيق. | - |
| **Django** | 5.x | إطار العمل الرئيسي لبناء منطق الأعمال. | يجب الالتزام بنمط **MVT (Model-View-Template)** لفصل الاهتمامات. |
| **Gunicorn** | 최신 | خادم تطبيقات WSGI لتشغيل Django في بيئة الإنتاج. | - |

## 3. الواجهة الأمامية (Frontend)

| التقنية | الإصدار | الغرض الأساسي | ملاحظات |
| :--- | :--- | :--- | :--- |
